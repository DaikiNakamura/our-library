# our-library

本を管理するアプリ！！  
Firebaseを前提に作ってます。

## Build Setup

``` bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev
```
※動作には`~/serviceAccountKey.json`が必要です。
``` json
{
  "apiKey": "",
  "authDomain": "",
  "databaseURL": "",
  "projectId": ""
}
```
